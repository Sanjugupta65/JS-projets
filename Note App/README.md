📝 Notes App
A simple and clean Notes App built with vanilla HTML, CSS, and JavaScript.
🚀 Features

Create notes with a title and body
Delete any note with one click
Notes are saved in localStorage — they stay even after page refresh
Basic validation — alerts user if fields are empty

🛠️ Built With

HTML5
CSS3
Vanilla JavaScript
localStorage API

📁 Project Structure
notes-app/
├── index.html
├── Note.css
└── Note.js
💡 What I Learned

DOM manipulation
Event listeners
localStorage to persist data
JavaScript arrays — push, forEach, splice
